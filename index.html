🏃 Path format reference

      

        Current default:
        $albumartist/$album (%left{$year,4})/$albumartist - $album - %right{00$track,2} - $title

        Example:
        WILLOW/WILLOW (2019)/WILLOW - WILLOW - 01 - Wait a Minute!.flac

        Useful variables:
        $albumartist $album $year $track $title $artist $disc $genre

        Useful functions:
        %left{x,n} %right{x,n} %aunique{} %lower{x} %if{cond,a,b}
